# Assam Political News Tracker

A specialized news aggregator for Assam that tracks political developments across 35 districts. Sourced from **India Today NE**, **The Sentinel**, and **Assam Tribune**.

## Features
- **District-wise Tracking**: 35 Districts (Baksa, Barpeta, etc.) + General.
- **Political Filtering**: Automatically filters for political keywords and excludes crime/accidents.
- **Sources**: Real-time headers from top regional papers.
- **Responsive UI**: Sidebar navigation for desktop, clean mobile view.

## Setup
1.  **Clone**: `git clone <your-repo-url>`
2.  **Install**: `npm install`
3.  **Run**: `npm run dev`
4.  **Deploy**: Push to GitHub and import into Vercel.

## Tech Stack
-   Next.js (App Router)
-   Cheerio (Scraping)
-   Tailwind CSS (Styling)
